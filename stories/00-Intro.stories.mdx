# Hahow Design System

A [Hahow for Business](https://business.hahow.in) React UI Components Library.

## Installation

npm:

```bash
npm install @hahow/hahow-design --save
```

or Yarn:

```bash
yarn add @hahow/hahow-design
```

## Usage

```jsx
import { Button } from '@hahow/hahow-design';

const App = () => (
  <Button>Click Me</Button>
);
```
